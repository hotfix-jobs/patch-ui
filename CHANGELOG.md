# Changelog

Release notes for the Patch UI copy-in registry.

## Unreleased

## 0.1.0 (2026-07-14)

### Added

- Initial versioned release of the Patch UI shadcn registry.
- React components, application blocks, shared recipes, utilities, and design
  tokens distributed through the `@patchui` namespace.
- Documentation, live previews, generated source references, search, and
  light and dark theme guidance at `ui.hotfix.jobs`.

### Distribution

- Patch UI is distributed as copied source through the shadcn registry.
- The former npm runtime package is retired. Private workspace packages exist
  only to build and verify the registry and documentation.
- The registry is versioned as one coordinated release rather than as
  individual components.
