# Tabs

Compound component built on Base UI Tabs. Two variants: `underline` (default) draws a single sliding underline that eases between the active tab position; `pill` renders each tab as an outlined rounded button with a filled active state. The underline animates via Base UI's `Tabs.Indicator` and the `--active-tab-*` CSS variables it populates. The `pill` variant works in both orientations, so one component can render a vertical sidebar on desktop and a horizontal strip on mobile.

## Usage

```tsx
import { Tabs, TabsList, TabsTrigger, TabsPanel } from "@patchui/react";

// Content tabs (underline)
<Tabs defaultValue="overview">
  <TabsList>
    <TabsTrigger value="overview">Overview</TabsTrigger>
    <TabsTrigger value="settings">Settings</TabsTrigger>
  </TabsList>
  <TabsPanel value="overview">...</TabsPanel>
  <TabsPanel value="settings">...</TabsPanel>
</Tabs>

// Nav-style, responsive: vertical pill on desktop, horizontal on mobile
<Tabs variant="pill" orientation="vertical" defaultValue="profile">
  <TabsList>
    <TabsTrigger value="profile">Profile</TabsTrigger>
    <TabsTrigger value="billing">Billing</TabsTrigger>
  </TabsList>
</Tabs>
```

## Components

- `Tabs` - root; accepts `variant`, `orientation`, `defaultValue` or controlled `value`/`onValueChange`
- `TabsList` - container for triggers; renders `Tabs.Indicator` in the underline variant
- `TabsTrigger` - individual tab; carries `data-[selected]` for active-state styling
- `TabsPanel` - matched panel for a trigger value

<Callout title="Notes">
- **Underline indicator:** Positioned via the `--active-tab-left` / `--active-tab-width` (or `top`/`height` when vertical) CSS variables that Base UI populates on `Tabs.Indicator`. Standard state/overlay easing tokens carry it between positions.
- **Underline vs pill:** Use `underline` for switching content panels; use `pill` for navigation-style tabs (settings sections, sidebar nav) where an outlined active state reads better.
- **Responsive nav:** With `variant="pill"`, render `orientation="vertical"` on larger screens and horizontal on small screens.
- **Keyboard:** Base UI ships proper roving tabindex + arrow-key navigation, Home / End, and standard focus outline.
</Callout>
