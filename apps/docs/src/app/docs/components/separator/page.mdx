# Separator

A hairline divider — horizontal or vertical — with an optional inline label for "OR" / "AND" splits.

<Install name="separator" />

<ComponentPreview label="Separator">
  <SeparatorDemo />
</ComponentPreview>

## Usage

```tsx
import { Separator } from "@patchui/react";

// Horizontal (default)
<Separator />

// With label
<Separator label="or" />

// Vertical (inside a flex row)
<div className="flex h-8 items-center gap-4">
  <span>Item A</span>
  <Separator orientation="vertical" />
  <span>Item B</span>
</div>
```

<Callout title="Notes">
- Vertical separators inherit their height from the parent — put them inside a `flex` (or `grid`) with a defined height, or they'll collapse.
- The label variant is horizontal only. It renders the hairline split around your text — good for "or" / "and" between two options.
- `decorative` is about semantics, not visuals. Turn it on when the divider is just a visual beat between blocks; leave it off when the divider marks a real section break the screen reader should announce.
</Callout>
