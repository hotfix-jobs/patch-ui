# Timeline

Horizontally paginated sequential content. One step visible at a time inside a full-width frame, with Back/Next buttons and a step counter in a hairline-divided footer row. All steps stay in the DOM so SEO crawlers and screen readers see the full content regardless of which one is on screen. Per-step background (color or image) crossfades on advance.

Intended for marketing surfaces: process explanations, brand chronicles, and similar ordered content.

## Usage

```tsx
import { Timeline } from "@patchui/react";
import { Bell, Radar, SlidersHorizontal } from "lucide-react";

<Timeline
  eyebrow="How it works"
  heading="From source to your inbox."
  steps={[
    {
      eyebrow: "01",
      icon: <Radar />,
      title: "We watch every source.",
      body: "Direct API pulls from every customer's hiring system.",
      details: ["Six integrations", "Live, not scraped", "Continuously refreshed"],
      meta: "Updates twice daily",
    },
    {
      eyebrow: "02",
      icon: <SlidersHorizontal />,
      title: "You set your preferences.",
      body: "Match on role, salary, skills, and location.",
    },
    {
      eyebrow: "03",
      icon: <Bell />,
      title: "We notify you.",
      body: "Push when matches land. Apply before everyone else.",
    },
  ]}
/>
```

## Props

- `steps`: `TimelineStep[]`. Required.
- `heading`: optional display heading rendered above the frame.
- `eyebrow`: optional small uppercase label rendered above the heading.

### TimelineStep

- `eyebrow`: small uppercase label rendered above the title (e.g. `"01"`, `"2022"`).
- `title`: step headline.
- `body`: optional supporting paragraph under the title.
- `icon`: optional decorative icon, typically a Lucide icon. Timeline standardizes the visual size.
- `details`: optional short bullet list rendered as hairline-divided rows. Use for sub-points, integrations, or stats.
- `meta`: optional small uppercase footer line below the details.
- `bg`: optional per-step background. `{ type: "color", value: "..." }` for any CSS color, or `{ type: "image", src: "..." }` for a cover-fit image.

## Interaction

- Touch: swipe horizontally across the frame.
- Keyboard: focus the timeline track and use Arrow Left / Right.
- Pointer: click the Back and Next buttons at the bottom of the frame.
