# SectionLabel

A small bold heading used to introduce a labeled section within a larger surface: sidebar nav groups, showcase-wall cells, demo variant clusters, foundations-page charts. Kept centralized so the whole app carries the same section-title voice.

## Usage

```tsx
import { SectionLabel } from "@patchui/react";

<div className="space-y-3">
  <SectionLabel>Bordered variants</SectionLabel>
  <VariantList />
</div>
```

<Callout title="Layout is the parent's job">
SectionLabel is typography only. It ships with no baked-in margin so no callsite has to fight a default. Reach for `space-y-*` or `gap-*` on the wrapper, or set explicit `mb-*` on the label itself when it makes sense.
</Callout>
