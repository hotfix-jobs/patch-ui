# Spinner

A single loading indicator. 12 tick marks arranged radially, each fading in sequence — a restrained rotating "clock hand" feel. Uses `currentColor`.

  <SpinnerDemo />

## Usage

```tsx
import { Spinner } from "@patchui/react";

// Default (md)
<Spinner />

// Sizes
<Spinner size="xs" />
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
<Spinner size="xl" />
<Spinner size="2xl" />
<Spinner size="3xl" />
<Spinner size="4xl" />

// Inherits text color
<div className="text-error">
  <Spinner />
</div>

// Custom a11y label for context
<Spinner label="Saving" />
<Spinner label="Uploading 3 of 12" />
```

<Callout title="Notes">
- One loading indicator across the system. Same silhouette everywhere so the loading language is consistent.
- Use `<Button loading>` — don't drop a Spinner inside a button label yourself. The Button wires the sizing and busy state for you.
- Use `variant="dots"` when the wait sits inline in copy ("Saving…"). Reach for `Skeleton` when async data will fill a known layout. Reach for `Progress` when total work is measurable.
- Colors inherit from `currentColor`. Wrap in a colored container to tint.
- Pure CSS animation, no JavaScript timer. Honors `prefers-reduced-motion` via the shared tokens.
- Set `label` to name the work in flight (`Verifying`, `Uploading 3 of 12`) so screen readers announce more than "Loading".
</Callout>
