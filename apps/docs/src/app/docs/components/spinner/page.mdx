# Spinner

A single loading indicator. 12 tick marks arranged radially, each fading in sequence — a restrained rotating "clock hand" feel. Uses `currentColor`.

## Usage

```tsx
import { Spinner } from "@patchui/react";

// Default (md)
<Spinner />

// Sizes
<Spinner size="xs" />
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
<Spinner size="xl" />

// Inherits text color
<div className="text-error">
  <Spinner />
</div>

// Custom a11y label for context
<Spinner label="Saving" />
<Spinner label="Uploading 3 of 12" />
```

<Callout title="Notes">
- **One spinner:** the system ships a single loading indicator. Use it everywhere loading needs to be shown, so the loading UX reads as one system.
- **Color inheritance:** the spinner uses `currentColor`, adapting to any text color. Wrap in a colored container to change the tint.
- **Button loading:** `<Spinner size="sm" />` fits inside a Button's loading state.
- **Pure CSS:** the animation runs entirely on the compositor thread. No JavaScript timers.
- **Accessibility:** renders with `role="status"` plus your `label` on `aria-label` and an sr-only span. Override `label` to surface specific async context to assistive tech.
</Callout>
