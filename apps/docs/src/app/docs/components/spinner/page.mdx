# Spinner

A single loading indicator. Rotating stroke arc on a low-opacity track. One variant, one silhouette, three sizes — the loading language stays consistent across every surface. Uses `currentColor` so it inherits the surrounding text tone.

## Usage

```tsx
import { Spinner } from "@patchui/react";

// Default (md)
<Spinner />

// Sizes — match the sm/md/lg vocabulary used across the system
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />

// Inherits text color
<div className="text-error">
  <Spinner />
</div>

// Custom a11y label for context
<Spinner label="Saving" />
<Spinner label="Uploading 3 of 12" />

// Larger loaders (page or empty-state) — override via className
<Spinner className="size-12" label="Loading dashboard" />
```

<Callout title="Notes">
- **One indicator, no variants.** Every loading state in the system uses the same rotating arc — no dots / bars / pulse variants — so the loading language stays consistent across surfaces.
- **Use `<Button loading>`.** Don't drop a Spinner inside a button label yourself. The Button wires the sizing and busy state for you.
- **Reach for other primitives when appropriate:** `Skeleton` when async data will fill a known layout, `Progress` when total work is measurable, `Spinner` for indeterminate short waits.
- **Colors inherit from `currentColor`.** Wrap in a colored container to tint.
- **Pure CSS animation, no JavaScript timer.** Honors `prefers-reduced-motion` via the shared tokens.
- **Set `label`** to name the work in flight (`Verifying`, `Uploading 3 of 12`) so screen readers announce more than "Loading".
</Callout>
