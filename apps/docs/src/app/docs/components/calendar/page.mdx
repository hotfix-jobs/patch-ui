# Calendar

Month grid for date selection. Three modes: `single`, `range`, `multiple`, with minimal chrome and tabular nums for clean date alignment. Use it standalone as a date picker inside any container (Modal, Sheet, Popover-style card) that you compose yourself.

## Usage

```tsx
import { Calendar, type DateRange } from "@patchui/react";

// Single date
const [date, setDate] = useState<Date | null>(null);
<Calendar mode="single" value={date} onValueChange={setDate} />

// Range
const [range, setRange] = useState<DateRange>({});
<Calendar mode="range" value={range} onValueChange={setRange} />

// Multiple
const [dates, setDates] = useState<Date[]>([]);
<Calendar mode="multiple" value={dates} onValueChange={setDates} />

// Constraints
<Calendar
  mode="single"
  fromDate={new Date("2026-01-01")}
  toDate={new Date("2026-12-31")}
  disabledDates={(d) => d.getDay() === 0 || d.getDay() === 6}  // disable weekends
/>

// Week starts on Monday (ISO)
<Calendar weekStartsOn={1} />
```

<Callout title="Notes">
- **Range mode flow:** First click sets `from` and clears `to`. Second click (after `from`) sets `to`. Clicking before `from` swaps them. Clicking the same day twice clears the selection.
- **Range visual:** Days between `from` and `to` get a subtle `--fill-1` background; the endpoints get the full `--primary` solid fill (inverted surface). Lets users see the range at a glance.
- **Today indicator:** Today's date renders with a hairline ring (`ring-1 ring-inset ring-hairline-strong`) when not selected.
- **Outside-month days:** Days that spill into the previous / next month are dimmed (opacity 60%) but remain clickable: clicking jumps the view to that month.
- **Tabular nums:** All numerals use `tabular-nums` so column alignment is perfect regardless of width.
</Callout>
