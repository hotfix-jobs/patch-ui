# Table

A set of composable table primitives - thin, styled wrappers around native table elements with hairline dividers, a surface background, and restrained typography. Mirrors the familiar shadcn table API so the parts compose the way you expect.

## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableRow,
  TableHead,
  TableCell,
  TableCaption,
} from "@patchui/react";

<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Plan</TableHead>
      <TableHead align="right">Price</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Team</TableCell>
      <TableCell align="right">$49</TableCell>
    </TableRow>
  </TableBody>
</Table>

// Flat variant — no radius, no outer bg, Bloomberg / NYT data-table feel
<Table variant="flat">
  ...
</Table>
```

<Callout title="Notes">
- **Composition:** `Table`, `TableHeader`, `TableBody`, `TableFooter`, `TableRow`, `TableHead`, `TableCell`, and `TableCaption` map directly to the native table elements, so standard HTML attributes pass through.
- **Default vs flat:** Pick `default` for dashboard / data-card surfaces (rounded, surface bg, contained). Pick `flat` when the table is the primary content of a longform layout: it sheds the outer chrome and lets the hairline grid + typography carry the table's structure.
- **Numerals:** Use `align="right"` on numeric `TableCell`s to right-align and enable tabular figures for clean column alignment.
- **Interactive rows:** Pass `interactive` to `TableBody` to add hover + cursor-pointer to every row. Hover fills the row with a `--surface-1` inset pill (rounded on the first/last cell) matching Menu item highlights, and drops the per-row hairlines since the pill is the row indicator.
- **Striped / bordered:** Opt in via `TableBody`: `striped` alternates even rows to `--surface-1` (also rounded inset), `bordered` adds vertical cell dividers.
- **Numerals:** Use `align="right"` on numeric `TableCell`s to right-align and enable tabular figures for clean column alignment.
- **TableHead typography:** Small caption tier (`text-caption-11 text-ink-muted`) for column headers, matching the caption pattern used in Command sections.
- **Key/value pairs:** For simple two-column key/value data, a two-cell `TableRow` per pair works well; reserve the header row for true tabular data.
</Callout>
