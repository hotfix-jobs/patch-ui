# Table

Composable native table primitives with clear horizontal dividers, restrained typography, and compact sorting controls.

<Install name="table" />

<ComponentPreview label="Table Example">
  <TableDemo />
</ComponentPreview>

## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableRow,
  TableHead,
  TableSortButton,
  TableCell,
  TableCaption,
  TableEmpty,
} from "@patchui/react";

// Default: open table with horizontal row dividers
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Plan</TableHead>
      <TableHead align="right">Price</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Team</TableCell>
      <TableCell align="right">$49</TableCell>
    </TableRow>
  </TableBody>
</Table>

// Optional outer boundary
<Table bordered>...</Table>

// Hover for actionable rows
<Table interactive>...</Table>

// Density
<Table size="sm">...</Table>

// Selected row
<TableRow selected onClick={...}>...</TableRow>

// Compact sort control
<TableHead aria-sort="ascending">
  <TableSortButton direction="asc" onClick={...}>Name</TableSortButton>
</TableHead>

// Sticky header inside a fixed-height scroll container
<div className="h-64 overflow-y-auto">
  <Table scrollable={false} stickyHeader>...</Table>
</div>

// Empty state
<TableBody>
  <TableEmpty colSpan={4}>No results.</TableEmpty>
</TableBody>
```

<Callout title="Notes">
- **Dividers are structural.** The header uses a stronger divider and every body row uses a soft divider. The last row drops its bottom divider.
- **Outer border is optional.** `bordered` adds a contained surface without changing row structure.
- **Composition:** Every sub-component maps to a native table element, so standard HTML attributes pass through.
- **Numerals:** Use `align="right"` on numeric `TableCell`s for right-align + tabular figures.
- **Sortable heads:** Compose `TableSortButton` inside `TableHead`, keep `aria-sort` on the header cell, and own the two-state or three-state cycle in your application.
- **Selected rows:** Combine with `interactive` on the root for the row-clickable pattern shown in the demo.
- **Sticky header:** Requires a fixed-height scroll container. Pass `scrollable={false}` so the internal horizontal-scroll wrapper doesn't fight your vertical-scroll parent.
- **Empty state:** Wrap `<EmptyState>` inside `<TableEmpty>` for the fullest treatment (icon + title + description + action). For a one-line placeholder, pass a string directly (`<TableEmpty colSpan={4}>No results.</TableEmpty>`).
</Callout>
