# Textarea

A multi-line text input. Auto-grows with content by default. Follows the same single pattern as `Input`: `background-100` fill, translucent border, 6px radius.

## Usage

```tsx
import { Textarea } from "@patchui/react";

// Basic
<Textarea placeholder="Write something…" />

// Custom minimum height
<Textarea placeholder="Tall textarea" rows={6} />

// Error state
<Textarea invalid value={value} onChange={…} />
```

<Callout title="Notes">
- **Auto-growing:** `field-sizing: content` grows the textarea with its content. The `rows` prop sets a minimum height.
- **Field integration:** internally wraps the native textarea with Base UI's `Field.Control` so it composes with `Field` for labels, descriptions, and validation messages.
- **Border stepping:** `gray-alpha-400 → gray-alpha-500` on hover → `gray-alpha-600` on focus, matching `Input`.
</Callout>
