# DisplayHeading

Big-display marketing typography. Three sizes, weight 500, tight letter-spacing scaled with size. Headings auto-balance line breaks via `text-wrap: balance` so multi-line titles distribute words evenly.

## Usage

```tsx
import { DisplayHeading } from "@patchui/react";

// Basic
<DisplayHeading size="xl">Build something people love.</DisplayHeading>

// With an eyebrow label above the heading
<DisplayHeading size="lg" eyebrow="Hiring">
  Sr. Engineer in NYC
</DisplayHeading>

// As an h2 instead of h1
<DisplayHeading render={<h2 />} size="md">
  Section title
</DisplayHeading>
```

## Props

- `size` - `"md" | "lg" | "xl"` (56 / 88 / 124px). Default `"lg"`.
- `eyebrow` - Optional small uppercase label rendered above the heading. Uses `--tracking-patch-label` and `--text-patch-micro` so it visually anchors a marketing display moment without competing.
- `render` - base-ui render prop, e.g. `<h2 />`. Default `<h1 />`.

<Callout title="Notes">
- **Line balancing:** Heading text is wrapped in a `text-wrap: balance` span so multi-line titles distribute words evenly across lines instead of leaving orphan tails on the last line.
- **Eyebrow positioning:** When present, the eyebrow renders as a block-display span inside the heading element (semantically valid). It carries its own micro-uppercase style and tertiary text color.
- **Data slots:** `display-heading`, `display-heading-eyebrow`, `display-heading-text` for granular styling.
</Callout>
