# Sheet

An edge-anchored panel that slides in from the top, right, bottom, or left. Two variants: `floating` (default) renders a rounded card inset 16px from the viewport edges so the backdrop peeks around it, right for editorial preview / inspector panels; `drawer` sits flush against the anchored edge with a single inward border and inward-facing corners rounded to match the floating card, right for mobile navigation and app-shell drawers. Pass `modal={false}` for a persistent inspector that keeps the page underneath interactive.

## Usage

```tsx
import {
  Button,
  Sheet,
  SheetTrigger,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetDescription,
  SheetBody,
  SheetFooter,
  SheetClose,
} from "@patchui/react";

<Sheet>
  <SheetTrigger render={<Button variant="secondary">Open profile</Button>} />
  <SheetContent>
    <SheetHeader>
      <SheetTitle>Profile</SheetTitle>
      <SheetDescription>Details for the selected member.</SheetDescription>
      <SheetClose />
    </SheetHeader>
    <SheetBody>
      {/* form / inspector content */}
    </SheetBody>
    <SheetFooter>
      <SheetClose render={<Button variant="secondary">Cancel</Button>} />
      <SheetClose render={<Button variant="primary">Save</Button>} />
    </SheetFooter>
  </SheetContent>
</Sheet>
```

<Callout title="Notes">
- **Two variants.** `floating` is a rounded card inset from every viewport edge (backdrop peeks around all four corners), right for inspectors and preview panels. `drawer` sits flush against the anchored edge, with the inward-facing corners rounded to match floating's radius, right for mobile navigation and app-shell drawers. Default is `floating`.
- **Modal by default.** Backdrop appears, page underneath is paused. Pass `modal={false}` for a persistent inspector (record detail, help drawer) that keeps the page interactive.
- **Outside-click, Escape, and `<SheetClose>` all dismiss.** For sheets holding unsaved input, wrap Save/Cancel in `SheetClose` and gate the dismiss with a confirmation step upstream.
- **Side matters.** Right for detail inspectors, left for navigation, bottom for mobile-first pickers. Keep the direction consistent across similar patterns in the app.
- **Focus return.** Focus returns to the trigger on close.
</Callout>
