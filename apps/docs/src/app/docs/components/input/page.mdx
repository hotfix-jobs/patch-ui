# Input

A text input aligned with Geist. Three sizes, styled/unstyled `prefix` + `suffix` slots, built-in `label` and `error` message, and a `rounded` (pill) variant. Uses `background-100` fill and translucent borders that step through hover/focus.

## Usage

```tsx
import { Input } from "@patchui/react";
import { Search, Mail, ArrowUpCircle } from "lucide-react";

// Basic
<Input placeholder="Enter text" />

// Sizes
<Input size="sm" placeholder="Small" />   // 32px
<Input size="md" placeholder="Medium" />  // 40px (default)
<Input size="lg" placeholder="Large" />   // 48px

// Prefix and suffix (styled by default — own container + separator)
<Input prefix={<ArrowUpCircle />} placeholder="Default" />
<Input suffix={<ArrowUpCircle />} placeholder="Default" />
<Input prefix="https://" suffix=".com" placeholder="Default" />

// Unstyled prefix/suffix (icons float inside the input area)
<Input prefix={<Search />} prefixStyling={false} placeholder="Search" />

// Inline label and error message
<Input id="domain" label="Domain" placeholder="example.com" />
<Input id="email" label="Email" error="Enter a valid email address." />

// Rounded (pill-shaped)
<Input rounded prefix={<Search />} prefixStyling={false} placeholder="Search" />

// Loading — disables input, shows spinner
<Input loading value={value} />

// For clearable search, use SearchInput (clears on Esc key or Esc chip click)
<SearchInput
  value={query}
  onChange={(e) => setQuery(e.target.value)}
  placeholder="Search projects"
/>
```

<Callout title="Notes">
- The `error` prop takes a boolean (just the visual state) or a string (renders the message inline with `role="alert"`). Prefer strings.
- Whitespace-trim the value at submit time so a stray space doesn't slip past a required check.
- Placeholders should look like a real value the user would type — `example.com`, `my-project`. Skip the "Enter your…" instructions; the label already does that job.
- Styled prefixes and suffixes read as their own module — good for `https://` schemes, `.com` domains, currency units. Turn styling off (`prefixStyling={false}`) when the affix is really an icon that lives inside the field.
- For an action button that sits inside an input row, use `<Button shape="circle">` with `aria-label`, not a naked icon.
- Use `<SearchInput>` for search boxes so keyboard Esc and the Esc chip both clear the value.
</Callout>
