# Tooltip

A popup that displays additional information when hovering over an element. Supports configurable placement and offset.

<Install name="tooltip" />

<ComponentPreview label="Tooltip Examples">
  <TooltipDemo />
</ComponentPreview>

## Usage

```tsx
import { Button, Tooltip, TooltipProvider } from "@patchui/react";

// Wrap tooltips in a provider for shared delay behavior
<TooltipProvider>
  {/* Basic tooltip */}
  <Tooltip content="Helpful information">
    <Button>Hover me</Button>
  </Tooltip>

  {/* Custom side */}
  <Tooltip content="Save your changes" side="bottom">
    <Button>Save</Button>
  </Tooltip>
</TooltipProvider>
```

<Callout title="Notes">
- Wrap a rail of tooltips in `<TooltipProvider>` so moving between adjacent triggers opens each one without the delay firing again. Tune the group delay on the provider.
- Tooltips open on hover *and* keyboard focus so keyboard users don't miss the hint. Any focusable trigger (Button, link, icon-only button) works.
- The popup renders through a portal so it's not clipped by nearby `overflow-hidden` containers.
- Keep the content short, a sentence at most. If the tip needs a paragraph, promote it to a `Modal` or `Sheet`.
</Callout>
