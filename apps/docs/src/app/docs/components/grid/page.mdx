# Grid

A decorative layout primitive with visible hairline guides between cells. Reach for it when the grid structure itself is part of the design: marketing pages, documentation landings, feature grids, showcase walls.

The hairlines are drawn with the `gap-px + colored parent background` trick. A single 1px `gray-alpha-400` gutter shows through between cells, and each `GridCell` covers the parent background with an opaque fill except in the gutters. Inspired by Vercel Geist's Grid.

## Usage

```tsx
import { Grid, GridCell } from "@patchui/react";

// Basic
<Grid columns={3}>
  <GridCell>Auto</GridCell>
  <GridCell>Auto</GridCell>
  <GridCell>Auto</GridCell>
</Grid>

// Responsive
<Grid columns={{ sm: 1, md: 2, lg: 3 }}>
  {items.map((i) => <GridCell key={i.id}>{i.label}</GridCell>)}
</Grid>

// Cell spanning two columns
<Grid columns={3}>
  <GridCell column="1 / 3">Wide</GridCell>
  <GridCell>Third</GridCell>
</Grid>

// Guides off (plain responsive columns)
<Grid columns={3} guides={false} className="gap-3">
  <GridCell>…</GridCell>
</Grid>
```

<Callout title="When not to use Grid">
For plain responsive content that doesn't need visible guides, reach for Tailwind's `grid grid-cols-*` utilities directly. Grid is only worth its weight when the hairline dividers are part of the visual design.
</Callout>
