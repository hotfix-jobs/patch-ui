# Button

A polymorphic button component with five variants, four sizes, icon support, three corner shapes, and a loading state. Aligned to Vercel Geist's Button spec.

## Usage

```tsx
import { Button } from "@patchui/react";

// Basic
<Button variant="primary">Deploy Project</Button>

// Variants
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="warning">Warning</Button>
<Button variant="error">Error</Button>

// Sizes: tiny is for icon-only rails
<Button size="tiny" aria-label="Add" icon={<Plus />} />
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>

// Icon-only + shape (aria-label required)
<Button shape="square" aria-label="Add" icon={<Plus />} />
<Button shape="circle" aria-label="Add" icon={<Plus />} />

// Rounded marketing shape
<Button shape="rounded" variant="primary" icon={<ArrowRight />} iconPosition="right">
  Start Free Trial
</Button>

// Loading state
<Button loading>Saving…</Button>

// As a link (polymorphic render prop)
<Button render={<a href="/dashboard" />}>Go to Dashboard</Button>
```

<Callout title="Notes">
- **Primary** fills with `gray-1000`. Reserve for the single hero action on a view.
- **Secondary** is `background-100` with a translucent border — the workhorse alternative.
- **Tertiary** is transparent with a subtle hover tint — low-emphasis actions.
- **Warning** is solid `amber-700` — reserve for actions with caution or reversible risk.
- **Error** is solid `red-700` — reserve for destructive actions.
- **Icon-only:** pass `icon` without `children`. The width becomes square (or circular / rounded) to match the size. **`aria-label` is required** — a dev warning fires without it.
- **`shape="rounded"` + marketing:** the 12px radius shape pairs with hero CTAs on marketing pages.
- **Polymorphic:** pass `render={<a href="…" />}` to render as an anchor while keeping button styles.
- **Loading state:** the button content is replaced with a spinner and the button is disabled to prevent double-submission.
- **Touch targets:** all sizes include a minimum 44×44px touch target via a pseudo-element.
- **Content:** Title Case, verb + noun (`Deploy Project`, `Delete Member`). Avoid bare `Submit` or `OK`.
</Callout>
