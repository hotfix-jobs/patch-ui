# Button

A polymorphic button component with five variants, four sizes, icon support, three corner shapes, and a loading state.

## Usage

```tsx
import { Button } from "@patchui/react";

// Basic
<Button variant="primary">Deploy Project</Button>

// Variants
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="warning">Warning</Button>
<Button variant="error">Error</Button>

// Sizes: tiny is for icon-only rails
<Button size="tiny" aria-label="Add" icon={<Plus />} />
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>

// Icon-only + shape (aria-label required)
<Button shape="square" aria-label="Add" icon={<Plus />} />
<Button shape="circle" aria-label="Add" icon={<Plus />} />

// Rounded marketing shape (typically paired with shadow)
<Button shape="rounded" shadow variant="primary" icon={<ArrowRight />} iconPosition="right">
  Start Free Trial
</Button>

// Dismissible pill (repo chip pattern — click opens, × removes)
<Button
  size="sm"
  shape="pill"
  variant="secondary"
  icon={<GitBranch />}
  onClick={openTag}
  onRemove={removeTag}
>
  design
</Button>

// Loading state
<Button loading>Saving…</Button>

// As a link (polymorphic render prop)
<Button render={<a href="/dashboard" />}>Go to Dashboard</Button>
```

<Callout title="Notes">
- Each variant maps to a specific token: `primary` = `gray-1000`, `secondary` = `background-200` + `gray-alpha-400` border (elevates over the page), `tertiary` = transparent + hover tint, `warning` = `amber-700`, `error` = `red-700`.
- Reserve `primary` for the one hero action on a view. Reserve `error` for actions the user cannot undo without work.
- Icon-only buttons: pass `icon` with no `children`. The button squares up to match the size. `aria-label` is required; a dev-time warning fires when it's missing.
- `shape="rounded"` alongside `shadow` produces the taller marketing pill (12px radius + soft lift) used on hero CTAs.
- Polymorphism: pass `render={<a href="…" />}` (or a `<Link>`) to render as an anchor with the button's styling and behavior.
- `loading` swaps the content for a spinner and disables interaction so a double-tap can't fire twice.
- Every size ships a hidden 44×44 touch target underneath, so the tap area works on coarse pointers even for a small button.
- Labels: Title Case, verb + noun (`Deploy Project`, `Delete Member`). A lone `Submit` or `OK` tells the user nothing about what will happen.
</Callout>
