# ToggleGroup

A compact segmented radio-style control. Use for view-mode toggles (grid / list), sort direction, density, or any small set of mutually-exclusive options.

<Install name="toggle-group" />

## Usage

```tsx
import {
  ToggleGroup,
  ToggleGroupItem,
} from "@patchui/react";
import { SquaresFour, Rows } from "@phosphor-icons/react/dist/ssr";

// Icon-only view toggle
<ToggleGroup value={view} onValueChange={setView} aria-label="View">
  <ToggleGroupItem value="grid" aria-label="Grid view">
    <SquaresFour />
  </ToggleGroupItem>
  <ToggleGroupItem value="list" aria-label="List view">
    <Rows />
  </ToggleGroupItem>
</ToggleGroup>

// Text labels
<ToggleGroup value={sort} onValueChange={setSort} aria-label="Sort">
  <ToggleGroupItem value="newest">Newest</ToggleGroupItem>
  <ToggleGroupItem value="oldest">Oldest</ToggleGroupItem>
</ToggleGroup>

// Small size for tight chrome
<ToggleGroup size="sm" value={view} onValueChange={setView} aria-label="View">
  ...
</ToggleGroup>
```

<Callout title="Notes">
- Reach for ToggleGroup when 2 to 3 options are mutually exclusive and different views of the same surface: `Grid` / `List`, `Source` / `Output`, alignment picker. Past 3 options, use `Tabs` or a `Select`.
- Icon-only items need their own `aria-label`: the icon alone doesn't announce the choice.
- Base UI ToggleGroup ships proper keyboard navigation (arrows / Home / End) and roving tabindex out of the box.
- For multi-select behavior, reach for a checkbox group instead — ToggleGroup here is intentionally single-select.
</Callout>
