# Avatar

A user avatar built on Base UI's Avatar. Shows an image and gracefully falls back to a monogram while the image loads or if it fails. Five sizes from `xs` to `xl`, `circle` or `square`. Compose overlapping rows with `AvatarGroup`.

## Usage

```tsx
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarGroup,
} from "@patchui/react";

<Avatar>
  <AvatarImage src="/people/jane.jpg" alt="Jane Doe" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>

// Sizes
<Avatar size="xs" />  // 24px — inline
<Avatar size="sm" />  // 28px
<Avatar size="md" />  // 36px (default)
<Avatar size="lg" />  // 44px
<Avatar size="xl" />  // 56px — profile hero

// Square
<Avatar shape="square">
  <AvatarImage src="/logo.png" alt="Acme" />
  <AvatarFallback>A</AvatarFallback>
</Avatar>

// Overlapping group with a +N chip
<AvatarGroup size="md" max={4}>
  <Avatar><AvatarFallback>JD</AvatarFallback></Avatar>
  <Avatar><AvatarFallback>AS</AvatarFallback></Avatar>
  <Avatar><AvatarFallback>MK</AvatarFallback></Avatar>
  <Avatar><AvatarFallback>LF</AvatarFallback></Avatar>
  <Avatar><AvatarFallback>BV</AvatarFallback></Avatar>
  <Avatar><AvatarFallback>RT</AvatarFallback></Avatar>
  {/* 6 children, max=4 → 4 avatars + "+2" chip */}
</AvatarGroup>
```

## Components

- `Avatar` — the container with size and shape variants
- `AvatarImage` — the image; covers the container and hides itself on error
- `AvatarFallback` — shown while the image loads or on error; typically initials
- `AvatarGroup` — overlapping row with auto size/shape propagation, hairline ring, and `+N` overflow chip

<Callout title="Notes">
- **Graceful fallback:** `AvatarFallback` renders until the image successfully loads, so there's no broken-image flash.
- **Group ring:** `AvatarGroup` adds a `ring-background-100` between avatars so they visually separate despite overlapping.
- **Overflow chip:** the `+N` chip uses `bg-gray-100` + `text-gray-900` so it reads as a counter, not another person.
</Callout>
