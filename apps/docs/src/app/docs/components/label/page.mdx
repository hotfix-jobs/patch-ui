# Label

A form label with the standard label typography (`label-14`, `gray-1000`). Connects to inputs via the native `htmlFor` attribute.

<Install name="label" />

<ComponentPreview label="Label Examples">
  <LabelDemo />
</ComponentPreview>

## Usage

```tsx
import { Label, Input } from "@patchui/react";

// Basic
<Label>Email address</Label>

// With an input
<Label htmlFor="email">Email</Label>
<Input id="email" type="email" />

// As a different element (polymorphic render prop)
<Label render={<span />}>Read-only label</Label>
```

<Callout title="Notes">
- **Typography:** `text-label-14` at `text-gray-1000` — reads as primary form chrome.
- **Polymorphic:** `render={<span />}` renders a `<span>` when you want the label style without the semantic tag.
- **Accessibility:** always pair with `htmlFor` matching the input's `id` so screen readers associate the label with its control.
</Callout>
