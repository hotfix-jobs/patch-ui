# Switch

A binary on/off slider for settings that take effect immediately. Three sizes, four intent variants for the on-state fill, and optional icons inside the thumb per state.

## Usage

```tsx
import { Switch } from "@patchui/react";
import { Lock, Unlock } from "lucide-react";

// Uncontrolled
<Switch defaultChecked />

// Controlled
const [checked, setChecked] = useState(false);
<Switch checked={checked} onCheckedChange={setChecked} />

// Sizes
<Switch size="sm" />
<Switch size="md" />  // default
<Switch size="lg" />

// Variants (on-state fill color)
<Switch variant="success" defaultChecked />
<Switch variant="warning" defaultChecked />
<Switch variant="error" defaultChecked />

// Icons inside the thumb, one per state
<Switch
  variant="success"
  icon={{ checked: <Lock />, unchecked: <Unlock /> }}
/>

// Disabled
<Switch disabled />
```

<Callout title="Notes">
- Reach for Switch when the state is a setting that saves the moment the user flips it — `Dark mode`, `Enable notifications`. Use `Checkbox` when the value only commits with a form submit.
- The on-state color signals what happens: default gray for a plain toggle, warning/error when turning it on has consequence (`Enable public access`).
- Icons inside the thumb make the state readable without a label — a lock/unlock pair, sun/moon, on/off. Icons inherit `text-gray-900` and size to ~60% of the thumb.
- Rendered as a `<button role="switch">` on top of a hidden `<input type="checkbox">`, so both keyboard users and native forms work as expected.
- For the toolbar press-button pattern (Bold / Italic / Star), reach for `Toggle` — it's a different behavior with `aria-pressed` state instead of a slider.
</Callout>
