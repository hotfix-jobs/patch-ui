# Badge

A small status indicator for labels, counts, and tags. Five semantic variants, two sizes, two shapes, and a dismissible pattern for filter chips.

## Usage

```tsx
import { Badge } from "@patchui/react";

// Basic
<Badge>Default</Badge>

// Semantic (solid)
<Badge variant="success">Active</Badge>
<Badge variant="warning">Pending</Badge>
<Badge variant="error">Failed</Badge>

// Outline (transparent + colored border)
<Badge variant="default" outline>Default</Badge>
<Badge variant="success" outline>Active</Badge>
<Badge variant="warning" outline>Pending</Badge>
<Badge variant="error" outline>Failed</Badge>

// Sizes
<Badge size="sm">Small</Badge>
<Badge size="lg">Large</Badge>

// Shape
<Badge shape="rounded">Rounded</Badge>
<Badge shape="pill">Pill</Badge>

// As a link
<Badge render={<a href="/status" />}>Linked</Badge>

// Dismissible — trailing × for filter chips, tokens, facets
<Badge onRemove={() => removeFilter("Design")}>Design</Badge>
```

<Callout title="Notes">
- **Contract:** every variant uses a hue-100 fill + hue-900 text. Contrast is verified in `check:contrast` at build time.
- **Polymorphic:** `render={<a href="…" />}` renders as an anchor while keeping badge styles.
- **Dismissible:** the × sits at 50% opacity, fades to 80% on badge hover, 100% on × hover.
- **Solid accent reserved:** stick to `success` / `warning` / `error` for actual state; don't reach for them decoratively.
</Callout>
